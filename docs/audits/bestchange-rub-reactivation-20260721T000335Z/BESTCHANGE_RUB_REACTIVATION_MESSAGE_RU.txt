Мы завершили повторную очистку и сертификацию RUB-направлений Exswaping после инцидента ZEC→SBP.

Подтверждённые изменения:
1) Для всех направлений в RUB теперь обязателен свежий независимый рыночный baseline. Проверка применяется одинаково к котировке, заявке и XML.
2) 32 направления без утверждённого baseline удалены из публичного XML и не включены в заявку на восстановление.
3) Внутренние банковские варианты CARDAMD сохранены, но в публичном XML оставлено только одно каноническое направление для каждой пары; дубликаты удалены.
4) В приложенном списке 69 направлений. Каждое имеет класс PASS либо PASS_EXPLAINED_SPREAD, подтверждённый резерв, разрешено для заявок и присутствует в публичном XML.
5) Публичный XML доступен по https://exswaping.com/currencies.xml: 20/20 последовательных проверок, без таймаутов и пустых ответов, XML валиден.

Просим повторно проверить Exswaping и восстановить только направления из приложенного списка. Если конкретная пара всё ещё не соответствует требованиям BestChange, пожалуйста, сообщите её код — мы разберём её отдельно по независимому baseline и политике премии.
